# Interactive Restaurant Finder App

## Overview
Interactive web application that allows users to discover restaurants using an interactive map and dynamic filtering system. The application enables users to explore locations, apply filters, and interact with map-based UI components in real time.

---

## Features
- Interactive map visualization with restaurant markers  
- Real-time filtering of restaurant data  
- Distance-based calculations for location discovery  
- Custom UI widgets for user interaction  
- Responsive and modular interface design  

---

## Architecture
- Model–View–Controller (MVC) architecture  
- Event-driven UI design  
- Modular and reusable component structure  
- Separation of concerns between data, logic, and presentation layers  

---

## Technologies Used
- TypeScript  
- HTML5 / CSS3  
- Vite  
- SimpleKit (UI framework)  
- JavaScript (event handling and logic)
